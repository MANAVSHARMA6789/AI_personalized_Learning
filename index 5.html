Process Flow

How SkillBridge Works

A four-stage pipeline that transforms raw resumes into actionable career intelligence in seconds.

📄

Resume Ingestion

Upload PDF/DOCX resume. PyPDF2 & spaCy extract raw text, tokenize entities, and identify named skills, experiences, and education with high accuracy NLP parsing.

👆 Click to see technical details
PyPDF2 strips text from PDF pages; python-docx handles DOCX format with layout preservation
spaCy en_core_web_lg model tokenizes and runs Named Entity Recognition (NER) for PERSON, ORG, DATE, and custom SKILL labels
Custom entity ruler patterns match 1,200+ tech skills, certifications and tools from a curated vocabulary
Extracted data is structured into: skills[], experience[], education[], and projects[] JSON fields
PyPDF2 3.0 spaCy 3.7 python-docx NER Entity Ruler
🗂️

Job Role Mapping

Select a target job role. The system loads a curated skill taxonomy for that role (e.g. Data Scientist: Python, ML, SQL, Statistics) from the structured knowledge base.

👆 Click to see technical details
240+ job role templates stored as weighted skill taxonomies (role → [{skill, weight, category}])
Each skill has a criticality weight (1–10) and a category tag (core / preferred / bonus)
Role templates sourced from LinkedIn job postings, Naukri.com JDs, and industry surveys (2023–2024)
Experience level (Fresher / Junior / Mid) adjusts required proficiency thresholds per skill
JSON Taxonomy SQLite KB 8 Roles Weighted Skills
🔍

Gap Analysis

NLP-driven cosine similarity and rule-based matching algorithms compare extracted student skills against role requirements, scoring each skill gap by criticality.

👆 Click to see technical details
TF-IDF vectorisation converts skill lists to vectors; cosine similarity scores semantic closeness (e.g. "ML" ≈ "Machine Learning")
Rule-based exact + fuzzy matching (difflib SequenceMatcher) handles abbreviations and synonyms
Gap score formula: gap_score = (1 − match_ratio) × skill_weight → aggregated into an overall readiness percentage
Gaps classified as: ✗ Missing (<40%), ~ Partial (40–70%), ✓ Present (>70%)
TF-IDF Cosine Sim difflib scikit-learn 94% Accuracy
🎓

Smart Recommendations

A weighted recommendation engine suggests curated learning resources (courses, certifications, projects) ranked by gap criticality, estimated ROI, and time-to-complete.

👆 Click to see technical details
Recommendation score = (skill_weight × gap_severity) + platform_quality_score + recency_bonus
3,800+ learning resources tagged by skill, platform, difficulty, duration and cost (free/paid)
Content-based filtering matches resource skill tags to identified gap skills using TF-IDF vectors
Supports 5 platforms: Coursera, Udemy, YouTube, LinkedIn Learning, and free open-source courses
Content-Based Filtering 3,800+ Resources 5 Platforms
📊

Analytics & Insights

Placement coordinators access an admin dashboard with batch analytics — cohort skill trends, placement readiness scores, and department-wise gap heatmaps.

👆 Click to see technical details
Flask REST API delivers aggregated cohort stats: avg readiness, top gap skills, dept breakdown
Chart.js 4 renders 6 chart types: line trend, doughnut, bar, horizontal bar, radar, and pie
Heatmap shows gap severity (%) across 5 roles × 7 skills — colour-coded red intensity scale
Student table supports sort and filter; each row links to individual gap report
Chart.js 4 Flask API Heatmap Real-time
